# CEF4Delphi
CEF4Delphi is an open source project created by Salvador DÃ­az Fau to embed Chromium-based browsers in applications made with Delphi.

CEF4Delphi is based on DCEF3, made by Henri Gourvest. The original license of DCEF3 still applies to CEF4Delphi. Read the license terms in the first lines of any *.pas file.

CEF4Delphi uses CEF 3.3359.1774.gd49d25f which includes Chromium 66.0.3359.181. 
The CEF3 binaries used by CEF4Delphi are available for download at spotify :
* [32 bits](http://opensource.spotify.com/cefbuilds/cef_binary_3.3359.1774.gd49d25f_windows32.tar.bz2)
* [64 bits](http://opensource.spotify.com/cefbuilds/cef_binary_3.3359.1774.gd49d25f_windows64.tar.bz2)


CEF4Delphi was developed and tested on Delphi 10.2 Tokyo and it has been tested in Delphi 7, Delphi XE, Delphi 10 and Lazarus 1.8.2/FPC 3.0.4. CEF4Delphi includes VCL, FireMonkey (FMX) and Lazarus components.
