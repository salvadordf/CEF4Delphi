type ICefResourceBundle = interface(ICefBaseRefCounted)

Description

Interface used for retrieving resources from the resource bundle (*.pak) files loaded by CEF during startup or via the ICefResourceBundleHandler returned from ICefApp.GetResourceBundleHandler. See TCefSettings for additional options related to resource bundle loading. The functions of this interface may be called on any thread unless otherwise indicated.

function GetDataResource(resourceId: Integer): ICefBinaryValue;

Returns a ICefBinaryValue containing the decompressed contents of the specified scale independent |resource_id| or NULL if not found. Include cef_pack_resources.h for a listing of valid resource ID values.

function GetDataResourceForScale(resourceId: Integer; scaleFactor: TCefScaleFactor): ICefBinaryValue;

Returns a ICefBinaryValue containing the decompressed contents of the specified |resource_id| nearest the scale factor |scale_factor| or NULL if not found. Use a |scale_factor| value of SCALE_FACTOR_NONE for scale independent resources or call GetDataResource instead.Include cef_pack_resources.h for a listing of valid resource ID values.
